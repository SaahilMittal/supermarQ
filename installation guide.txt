The artifact is available
on Zenodo (10.5281/zenodo.5786391). The
source code and artifact notebook are zipped within
supermarq_hpca_ae.tgz.

The supermarq software package and all of its dependencies can be installed via:

```

cd SupermarQ_HPCA_Artifact
pip install -r requirements.txt
pip install -e .
pip install sklearn
pip install qiskit
pip install supermarq
pip install cirq
pip install notebook
pip install jupyter lab


The jupyter notebook can be started with:

```
jupyter notebook

```
